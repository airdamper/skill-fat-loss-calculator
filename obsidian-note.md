# 减脂增肌计算器 — Agent 技能笔记

## 技能概述

本技能用于根据用户提供的身体参数（体重、体脂率可选、目标、活动水平），**自动计算每日宏量营养素摄入目标**，并可选地**生成一周训练计划模板**。

所有计算规则严格引用 `[[减脂增肌技能手册]]`，不允许自行修改公式或系数。

---

## 第一步：参数收集

与用户确认以下信息（如果用户未给出全部，请主动追问）：

| 参数 | 备注 |
| :--- | :--- |
| 体重(kg) | 必填，示例：70 |
| 体脂率(%) | 可选，若提供则优先使用瘦体重计算；若不提供则直接用体重 |
| 目标 | **必须为「减脂」「增肌」「维持」之一** |
| 活动水平 | **低**（久坐）、**中**（每周3-5次运动）、**高**（体力工作+每日训练） |

---

## 第二步：计算宏量营养素

### 2.1 蛋白质

| 目标 | 系数（克/公斤体重） | 备注 |
| :--- | :--- | :--- |
| 减脂 | 1.6 – 2.2 | 建议取用户体重×2.0（通用平衡值） |
| 增肌 | 1.6 – 2.0 | 建议取用户体重×1.8 |
| 维持 | 1.0 | 直接取1.0 |

> **如果用户提供了体脂率**，可以先计算瘦体重(kg) = 体重×(1-体脂率)，然后用瘦体重×2.2（减脂）或×2.0（增肌）得出更精确的蛋白质值。此时需向用户说明“基于瘦体重的精确推荐”。

### 2.2 碳水化合物

| 目标 | 系数（克/公斤体重） | 最低限制 |
| :--- | :--- | :--- |
| 减脂 | 2.0 | 不低于120克/天 |
| 增肌 | 3.0 – 5.0 | 体脂率高取低值，体脂率低取高值 |
| 维持 | 总热量的40%-50% | 无硬性下限 |

> **分配建议**（仅当用户要求详细餐单时提供）：  
> - 将大部分碳水集中在**练前和练后**两餐  
> - 非训练日降低碳水总量至2.0克/kg（减脂期）或2.5克/kg（增肌期）

### 2.3 脂肪

- **所有阶段**：摄入量 = **总热量的20% – 30%**
- 先根据蛋白质和碳水估算热量：
  - 蛋白质热量 = 蛋白质(g)×4
  - 碳水热量 = 碳水(g)×4
  - 总热量 = (蛋白质+碳水热量) ÷ (1-脂肪占比目标值)
  - 脂肪热量 = 总热量×（0.25 取中值）
  - 脂肪克数 = 脂肪热量 ÷ 9

**简算公式（当用户不需要精确热量时）**：
- 减脂期：脂肪 = 体重×0.5 – 0.8 克/公斤（最低不低于0.5克/kg，一般推荐0.6-0.8）
- 增肌期：脂肪 = 体重×0.8 – 1.0 克/公斤
- 向用户输出时优先使用 **克数**，避免直接输出热量（用户更直观）

---

## 第三步：输出格式

直接给用户呈现一个 **清晰表格**，例如（以70kg，减脂为例）：

| 营养素 | 每日摄入目标 | 备注 |
| :--- | :--- | :--- |
| 蛋白质 | 140g | 2.0g/kg，分3-4餐，每餐约35g |
| 碳水 | 140g | 2.0g/kg，练前练后占大部分 |
| 脂肪 | 45g | 约0.64g/kg，选择天然来源 |
| 估算总热量 | ≈1565 kcal | 蛋白质560+碳水560+脂肪405 |

**如果用户还要求食谱**，参考 `[[减脂增肌技能手册]]` 中的食物换算表组合一个示例（避免超过两个表格）。

---

## 第四步：生成训练计划（可选）

如果用户要求“训练计划”或“周计划”，则按以下模板输出：

### 减脂期周模板

| 星期 | 训练内容 | 备注 |
| :--- | :--- | :--- |
| 周一 | 力量·推（胸/肩/三头） | 复合动作优先 |
| 周二 | 低强度有氧 40min | 恒速，如快走 |
| 周三 | 力量·拉（背/二头） | 引体/划船 |
| 周四 | 低强度有氧 30min | 单车/椭圆机 |
| 周五 | 力量·腿+核心 | 深蹲/硬拉/箭步蹲 |
| 周六 | HIIT 20min 或 户外活动 | 工作20s+休息40s，4-6组 |
| 周日 | 主动恢复（慢走/拉伸） | 不计入训练量 |

### 增肌期周模板（去掉专项有氧，仅在力量训练后加15分钟有氧作为热身）

- 频率：每周4-5次力量训练，可上下肢分训或推拉腿分训
- 有氧：控制在每周2次以内，每次≤20分钟

> 如果用户提供具体数字（如“我周二没时间”），可微调顺序，但确保训练间隔合理。

---

## 第五步：调整建议与警告（必含）

在输出末尾添加以下标准文字（可根据用户情况选择）：

> **⚠️ 重要提醒**  
> 1. 体重短期波动正常，关注周平均趋势和腰围变化。  
> 2. 平台期可尝试碳水循环或增加有氧。  
> 3. 若出现关节痛、持续疲劳、情绪低落，请安排减载周（降低30-50%训练量）。  
> 4. 本计算基于通用公式，若有已知疾病（如糖尿病、肾脏问题）请先咨询医生。

---

## 技能调用的样例指令（供其他 agent 参考）

当用户在对话中说出以下指令时，请触发本技能：

- “帮我计算减脂期的营养素，体重75kg，体脂18%”  
- “增肌计划，80kg，中等活动水平”  
- “我70kg，想减脂，给个一周训练计划”  
- “计算我的蛋白质和碳水，目标是维持”

**调用后，直接按上述流程执行，不需要再询问“是否使用减脂增肌技能手册？”，默认本技能已被启用。**
