## 减脂增肌计算器 — Agent 技能说明（v2.1）

当用户提到以下触发词时，自动执行本技能：减脂、增肌、计算营养素、训练计划、减脂增肌、内脏脂肪、内脂、消内脏脂肪、维生素、缺维生素、疲劳没劲、运动表现下降。

### 执行规则

1. **收集参数**：
   - 必填：体重(kg)、目标(减脂/增肌/维持)
   - 可选（如果用户主动提供或你感觉合适时追问）：体脂率(%)、腰围(cm)、当前压力水平(1-10)、睡眠状况（小时）

2. **计算营养素**（与之前一致）：
   - 蛋白质：减脂 = 体重×2.0，增肌 = 体重×1.8，维持 = 体重×1.0。若用户提供体脂率，优先用瘦体重×2.2(减脂)×2.0(增肌)。
   - 碳水：减脂 = 体重×2.0（不低于120g），增肌 = 3.0-5.0（据体脂率调整），维持 = 总热量40%-50%。
   - 脂肪：总热量20%-30%。简算：减脂 = 体重×0.6-0.8，增肌 = 体重×0.8-1.0。

3. **输出格式**：
   - 基本表格（蛋白/碳水/脂肪/估算总热量）
   - **每条输出必须附带一句**：🍽️ 食物质量优先原则：每餐先吃蛋白质和脂肪（稳血糖），再吃碳水。稳定血糖比算热量更能自然降低食欲，是消减内脏脂肪的核心策略。

4. **可选生成训练计划**：
   - 先判断用户是否提供了压力水平或睡眠状况：
     - 如果用户反馈 **压力大（7-10）或睡眠＜6h** → 推荐 **低强度有氧为主**（可说话强度），避免HIIT和长时间无氧。**同时追加**：🧠 高压状态下身体消耗B族和维生素C更快，建议今日至少有一餐包含全谷物（燕麦/糙米）和一份彩色蔬菜/橙子，支持神经递质合成和皮质醇代谢。
     - 如果用户反馈 **压力一般或睡眠≥7h** → 按标准模板。
   - 标准减脂期模板：
     - 周一 力量推 | 周二 有氧40min | 周三 力量拉 | 周四 有氧30min | 周五 腿+核心 | 周六 HIIT20min | 周日 主动恢复

5. **当用户目标为「减脂」或提到「内脏脂肪」时，必须额外输出以下权重表**：

| 影响因素 | 权重 | 你的优化方向 |
| --- | --- | --- |
| 热量缺口 | ~30-35% | 通过食物质量自然达成，而非节食 |
| 食物质量（稳血糖） | ~25% | 每餐先蛋白+脂肪，后碳水 |
| 科学运动 | ~15-20% | 低强度有氧为主 + 每周2-3次抗阻/HIIT |
| 管理压力/皮质醇 | ~10% | 保证7小时睡眠，压力大时降低训练强度；同时注意补充B族维生素和维C |
| 减少酒精 | ~5% | 停止饮酒 |
| 类胡萝卜素+儿茶素 | ~2-3% | 每日一份彩色蔬菜 + 一杯纯抹茶 |

6. **当用户提到「疲劳」「没劲」「运动表现下降」「平台期」时，在对应回答后追加自查清单**：

> 🔍 **微量营养素自查**：
> - 每日是否有至少一顿**粗粮**（燕麦/糙米/全麦）？→ 缺则可能影响B1/B5摄入，降低能量利用效率
> - 每日是否有**150g以上瘦肉/蛋/鱼**？→ 缺则可能影响B6/B12/铁/锌供应
> - 是否有**足量户外日照10-15分钟**或补充维D？→ 缺则影响肌肉功能和钙吸收
> - 减脂期碳水偏低时，是否关注了B6摄入？→ 缺则影响蛋白质代谢和糖异生

7. **最后须附警告**（更新版）：
   - 体重短期波动正常，关注**周平均腰围**变化。
   - 平台期可尝试碳水循环或增加低强度有氧。
   - 若关节痛、持续疲劳、情绪低落，安排减载周（降低30-50%训练量）。
   - 若患糖尿病、肾脏问题等请先咨询医生。
   - **食物质量优先于卡路里**：天然食物合理份量比精确计算卡路里更可持续。
   - **微量营养素不可忽视**：长期节食或极低碳水可能导致B族缺乏，表现为易疲劳、运动表现下降。每日饮食尽量包含全谷物、深绿色蔬菜、瘦肉/蛋类，必要时可补充复合B族+维D。
