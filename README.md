# 💪 减脂增肌计算器 — 通用 Agent 技能包 v2.1

> 一个为 AI Agent（如 Obsidian Copilot、ChatGPT、Claude 等）设计的标准化技能包。  
> 根据用户的体重、体脂率、目标（减脂/增肌/维持）和活动水平，**自动计算每日宏量营养素摄入目标**，并可生成**一周训练计划模板**。  
> **v2.1 新增**：内脏脂肪消减因素权重分析、食物质量优先原则、基于压力/睡眠动态调整训练推荐、**微量营养素关联指导**（B族、维C、维D、铁、锌）。

---

## ✨ 功能特点

- **精准计算**：基于《减脂增肌技能手册》制定，蛋白质、碳水、脂肪三步到位
- **体脂适配**：可选体脂率参数，支持瘦体重优先计算
- **内脏脂肪管理**：输出消减内脏脂肪的关键因素权重表，涵盖热量缺口、食物质量、运动、压力管理、酒精、营养素补充
- **食物质量优先**：强调稳定血糖 > 计算卡路里，每餐先蛋白质和脂肪后碳水
- **压力/睡眠自适应训练**：根据用户压力水平或睡眠时长动态调整训练类型（高压时推荐低强度有氧，避免HIIT）
- **微量营养素关联指导**：整合维生素与矿物质知识库，在用户提到「疲劳」「压力大」「平台期」「运动表现下降」时，提供对照自查清单和补充建议，提升能量代谢效率
- **训练模板**：内置减脂期和增肌期周计划，涵盖力量、有氧、HIIT、主动恢复
- **即装即用**：支持作为 Obsidian 笔记、Agent System Prompt 或 API 端点部署
- **开源共享**：MIT 许可证，欢迎 Fork、PR 和个性化修改

## 📦 安装方式

### 🅰 作为 Obsidian 笔记（推荐）

1. 将 `obsidian-note.md` 内容复制到你的 Obsidian vault 中，新建笔记，命名为 `技能-减脂增肌计算器`
2. （可选）在 Obsidian Copilot 的 **Custom Prompt** 中加入以下触发规则：

当用户提到“减脂”、“增肌”、“营养素”、“训练计划”、“内脏脂肪”、“内脂”、“腰围”、“食物质量”、“维生素”、“疲劳没劲”、“平台期”等关键词时，请优先搜索并遵循笔记 技能-减脂增肌计算器 中的逻辑执行。

3. 现在直接对 AI 说“70kg，减脂”，即可自动调用技能。

### 🅱 作为通用 System Prompt

1. 打开你的 Agent（ChatGPT、Claude、Gemini 等）
2. 在设置中找到 System Prompt / Custom Instructions 区域
3. 将 `agent-prompt.md` 的全部内容粘贴进去
4. 保存即可。

### 🅲 通过 URL 远程加载（适合支持 URL 访问的 Agent）

在对话开始时发送：

请阅读 [https://raw.githubusercontent.com/airdamper/skill-fat-loss-calculator/main/obsidian-note.md](https://raw.githubusercontent.com/airdamper/skill-fat-loss-calculator/main/obsidian-note.md) 的内容，并按技能逻辑执行。我的参数是：体重70kg，减脂，腰围90cm，最近压力较大。

> 部分 Agent 不支持直接读取 raw 文件，请以方式 A 或 B 为主。

## 🧠 使用方法

### 触发指令示例

|场景|示例指令|
|:--|:--|
|计算营养素|“帮我算一下减脂期的营养素，体重75kg，体脂18%”|
|生成训练计划|“我70kg，想增肌，给个一周训练计划”|
|内脏脂肪咨询|“我腰围92cm，想消内脏脂肪，有什么建议？”|
|压力调整训练|“最近压力大，训练计划怎么调？”|
|简算|“计算蛋白质和碳水，目标是维持，体重80kg”|
|疲劳自查|“减脂期最近总是没劲，会不会缺什么？”|
|平台期排查|“体重两周不动了，是不是微量营养素不够？”|
|维生素咨询|“减脂要不要补维生素？”|
|详细咨询|“使用减脂增肌计算器，体重65kg，减脂，中等活动水平，压力7/10，睡眠不足6小时”|

### 输出示例（以 70kg 减脂为例）

|营养素|每日摄入目标|备注|
|:--|:--|:--|
|蛋白质|140g|2.0g/kg，分3-4餐|
|碳水|140g|2.0g/kg，练前练后为主|
|脂肪|45g|约0.64g/kg，天然来源|
|估算总热量|≈1565 kcal|仅供参考|

> 🍽️ 食物质量优先原则：每餐先吃蛋白质和脂肪（稳血糖），再吃碳水。稳定血糖比算热量更能自然降低食欲，是消减内脏脂肪的核心策略。

#### 当用户目标涉及「内脏脂肪」时，额外输出：

|影响因素|权重|优化方向|
|:--|:--|:--|
|热量缺口|~30-35%|通过食物质量自然达成|
|食物质量（稳血糖）|~25%|先蛋白+脂肪，后碳水|
|科学运动|~15-20%|低强度有氧为主 + 抗阻/HIIT|
|管理压力/皮质醇|~10%|保证7h睡眠，压力大时降低强度；注意补充B族和维C|
|减少酒精|~5%|停止饮酒|
|类胡萝卜素+儿茶素|~2-3%|每日彩色蔬菜 + 抹茶|

> ⚠️ 不存在单一捷径，低占比因素无法抵消核心诱因（高血糖、高皮质醇、热量盈余）。

#### 当用户提到「疲劳」「没劲」「运动表现下降」时，追加：

> 🔍 **微量营养素自查清单**：
> 
> - 每日是否有至少一顿**粗粮**（燕麦/糙米/全麦）？→ 缺B1/B5会降低能量利用效率
> - 每日是否有**150g以上瘦肉/蛋/鱼**？→ 缺B6/B12/铁/锌会影响能量代谢和氧输送
> - 是否有**足量户外日照10-15分钟**或补充维D？→ 缺维D影响肌肉功能和钙吸收
> - 减脂期碳水偏低时，是否关注了B6摄入？→ 缺B6影响蛋白质代谢和糖异生
> - 女性是否关注了铁摄入？→ 女性铁需求高于男性，缺铁表现为稍有运动就心率飙升

#### 当用户反馈「压力大」或「睡眠<6h」时，训练调整后追加：

> 🧠 **能量代谢辅助建议**：高压状态下身体消耗B族和维生素C更快。建议今日至少有一餐包含**全谷物**（补充B1、B5）和一份**彩色蔬菜/橙子**（补充维C），以支持神经递质合成和皮质醇代谢。

#### 当用户提到「平台期」或「体重不动」时，追加：

> 🔧 **检查微量营养素是否拖了后腿**：长期低碳或严格控制热量时，B族维生素（特别是B1、B6）摄入可能不足，直接影响能量代谢效率。建议确保每日有一顿粗粮和足量瘦肉/蛋类。

#### 当用户主动问「维生素」「要不要补」时，启动完整指导：

> **减脂期特别需要关注的微量营养素**：
> 
> - **B族维生素**：能量代谢关键辅酶，缺乏将降低运动表现，来源为全谷物、瘦肉、蛋类
> - **维生素D**：维持肌肉功能，来源为户外日照（10-15分钟/天）或补充剂
> - **铁**：有氧耐力的保障，女性尤其需要关注，来源为红肉、动物肝脏
> - **锌**：睾酮合成辅助因子，来源为生蚝、牛肉
> - **首选食物补充，补剂作为最后选择**

## 🔧 自定义指南

1. **修改系数**：如果你有自己的营养师建议，可以调整 `agent-prompt.md` 中的蛋白质系数（如减脂期改用 2.0→2.2）
2. **替换训练模板**：在 `obsidian-note.md` 第三步中替换你自己的周计划
3. **增加参数**：在 `skill.json` 的 `inputParameters` 中添加新字段（如“腰部受伤史”），并在推理逻辑中处理
4. **调整压力/睡眠阈值**：可修改 `agent-prompt.md` 中压力水平>7或睡眠<6h的判断条件
5. **扩展微量营养素关联**：可在 `obsidian-note.md` 第三步中增加更多自查项（如钙、镁、硒等），参考 `[[维生素与核心微量元素全维度拆解]]` 内容
