# 💪 减脂增肌计算器 — 通用 Agent 技能包 v2.3

> 跨平台 AI Agent 技能包（兼容 Reasonix / Claude Code / Codex / Qoder 等支持 Anthropic Skills Protocol 的工具）。  
> 根据用户的体重、体脂率、目标（减脂/增肌/维持）和活动水平，**自动计算每日宏量营养素摄入目标**，并可生成**一周饮食与训练计划**。  
> **v2.3 新增**：系统性平替方法——食物按营养密度等量替换（含参照表+公式+口味/预算/素食/过敏规则），训练按动作模式+难度+设备三级平替。

---

## ✨ 功能特点

- **精准计算**：基于《减脂增肌技能手册》制定，蛋白质、碳水、脂肪三步到位，支持瘦体重优先
- **内脏脂肪管理**：输出消减内脏脂肪的关键因素权重表，涵盖热量缺口、食物质量、运动、压力管理、酒精、营养素补充
- **食物质量优先**：强调稳定血糖 > 计算卡路里，每餐先蛋白质和脂肪后碳水
- **压力/睡眠自适应训练**：根据用户压力水平或睡眠时长动态调整训练类型（高压时推荐低强度有氧，避免HIIT）
- **饮食与训练平替**：支持食物等量替换（按营养密度换算，适配口味/预算/素食/过敏）与训练动作等量替换（六大动作模式×三级难度×三级设备）
- **微量营养素关联指导**：在用户提到「疲劳」「压力大」「平台期」「运动表现下降」时，提供对照自查清单和补充建议
- **训练模板**：内置减脂期正常版、减脂期高压版、增肌期版三套周计划，每项关联饮食要点
- **跨平台兼容**：一份 `SKILL.md`，到处运行
- **开源共享**：MIT 许可证，欢迎 Fork、PR 和个性化修改

## 📦 安装方式

本项目使用标准 **Anthropic Skills Protocol** 格式（`SKILL.md` + YAML frontmatter），以下是各平台安装方法。

### 🅰 Reasonix

```bash
# 从 GitHub 直接安装
reasonix install-source https://github.com/airdamper/skill-fat-loss-calculator
```

或在 `reasonix.toml` 的 `[skills]` 中添加本仓库路径。

安装后，在对话中提及相关触发词（减脂、增肌、计算营养素 等）即可自动调用。

### 🅱 Claude Code

将本仓库克隆到本地，在项目下创建 `.claude/skills/` 目录并建立软链接：

```bash
# 在 Claude Code 项目下
cd your-project
mkdir -p .claude/skills
cp /path/to/skill-fat-loss-calculator/SKILL.md .claude/skills/
```

或在 Claude Code 设置中指向本仓库路径。

### 🅲 Codex / Qoder

Codex CLI、Qoder 等工具同样遵循 `SKILL.md` 格式，方式类似：

1. 克隆本仓库
2. 将 `SKILL.md` 放置到工具的 skills 目录下
3. 或直接通过 URL 加载：要求 Agent 读取以下文件内容后按技能逻辑执行：

```
https://raw.githubusercontent.com/airdamper/skill-fat-loss-calculator/main/SKILL.md
```

### 🅳 通用 System Prompt（适用任何 AI）

将 `SKILL.md` 的全部内容（去除 YAML frontmatter）粘贴到 Agent 的 System Prompt / Custom Instructions 区域即可。

## 🧠 使用方法

### 触发指令示例

|场景|示例指令|
|:--|:--|
|计算营养素|"帮我算一下减脂期的营养素，体重75kg，体脂18%"|
|生成饮食与训练计划|"我70kg，减脂，给个一周饮食和训练计划"|
|内脏脂肪咨询|"我腰围92cm，想消内脏脂肪，有什么建议？"|
|压力调整训练|"最近压力大，训练计划怎么调？"|
|平替（素食/过敏/预算）|"不想吃鸡胸肉，换别的" / "学生党没器械怎么练"|
|疲劳自查|"减脂期最近总是没劲，会不会缺什么？"|
|平台期排查|"体重两周不动了，是不是微量营养素不够？"|
|维生素咨询|"减脂要不要补维生素？"|
|详细咨询|"使用减脂增肌计算器，体重65kg，减脂，中等活动水平，压力7/10，睡眠不足6小时，素食，居家"|

### 输出示例（以 70kg 减脂为例）

|营养素|每日摄入目标|备注|
|:--|:--|:--|
|蛋白质|140g|2.0g/kg，分3-4餐|
|碳水|140g|2.0g/kg，练前练后为主|
|脂肪|45g|约0.64g/kg，天然来源，含欧米伽3|
|膳食纤维|25-35g|1斤蔬菜+半斤粗粮|
|估算总热量|≈1565 kcal|仅供参考|

> 🍽️ 食物质量优先原则：每餐先吃蛋白质和脂肪（稳血糖），再吃碳水。稳定血糖比算热量更能自然降低食欲，是消减内脏脂肪的核心策略。

#### 当用户目标涉及「内脏脂肪」时，额外输出：

|影响因素|权重|优化方向|
|:--|:--|:--|
|热量缺口|~30-35%|通过食物质量自然达成|
|食物质量（稳血糖）|~25%|先蛋白+脂肪，后碳水|
|科学运动|~15-20%|低强度有氧为主 + 抗阻/HIIT|
|管理压力/皮质醇|~10%|保证7h睡眠，压力大时降低强度；注意补充B族和维C|
|减少酒精|~5%|停止饮酒|
|类胡萝卜素+儿茶素|~2-3%|每日彩色蔬菜 + 抹茶|

> ⚠️ 不存在单一捷径，低占比因素无法抵消核心诱因（高血糖、高皮质醇、热量盈余）。

#### 当用户提到「疲劳」「没劲」「运动表现下降」时，追加：

> 🔍 **微量营养素自查清单**：
> - 每日是否有至少一顿**粗粮**（燕麦/糙米/全麦）？→ 缺B1/B5会降低能量利用效率
> - 每日是否有**150g以上瘦肉/蛋/鱼**？→ 缺B6/B12/铁/锌会影响能量代谢和氧输送
> - 是否有**足量户外日照10-15分钟**或补充维D？→ 缺维D影响肌肉功能和钙吸收（维D需经皮肤→肝脏→肾脏三步活化）
> - 减脂期碳水偏低时，是否关注了B6摄入？→ 缺B6影响蛋白质代谢和糖异生
> - 女性是否关注了铁摄入？→ 女性铁需求高于男性，缺铁表现为稍有运动就心率飙升

#### 当用户反馈「压力大」或「睡眠<6h」时，训练调整后追加：

> 🧠 **能量代谢辅助建议**：高压状态下身体消耗B族和维生素C更快。建议今日至少有一餐包含**全谷物**（补充B1、B5）和一份**彩色蔬菜/橙子**（补充维C），以支持神经递质合成和皮质醇代谢。

#### 当用户提到「平台期」或「体重不动」时，追加：

> 🔧 **检查微量营养素是否拖了后腿**：长期低碳或严格控制热量时，B族维生素（特别是B1、B6）摄入可能不足，直接影响能量代谢效率。建议确保每日有一顿粗粮和足量瘦肉/蛋类。

#### 当用户主动问「维生素」「要不要补」时，启动完整指导：

> **减脂期特别需要关注的微量营养素**：
> - **B族维生素**（B1/B2/B3/B5/B6/B7/B9/B12）：能量代谢关键辅酶，来源为全谷物、瘦肉、蛋类、绿叶菜
> - **维生素D**：维持肌肉功能，促进钙吸收，来源为户外日照（10-15分钟/天）或维D3补剂
> - **铁**：有氧耐力的保障，女性尤其需要关注，来源为红肉、动物肝脏
> - **锌**：睾酮合成辅助因子，来源为生蚝、牛肉；注意植酸影响吸收
> - **维生素C**：抗氧化，彩椒含量远高于橙子
> - **首选食物补充，补剂作为最后选择**

## 🔄 平替功能说明（v2.3 新增）

本技能支持**食物**与**训练动作**的系统性等量替换，规则全部内置于 `SKILL.md` 中，模型可直接自动执行。

### 🥩 食物平替

当用户因口味偏好、预算、素食或过敏需要换食材时，模型会查**营养参照表**并用公式计算等价份量：

```
新食物重量 = 原食物重量 × (原食物蛋白密度 / 新食物蛋白密度)
```

| 使用场景 | 示例指令 | 技能自动执行 |
|:---------|:---------|:-------------|
| 口味偏好 | "不想吃鸡胸肉，换牛肉" | 查表：鸡胸肉31g蛋白/100g → 牛肉26g蛋白/100g，计算新重量 |
| 预算有限 | "三文鱼太贵了，换便宜的" | 预算平替：三文鱼→鲭鱼 |
| 素食 | "我吃素，不吃肉" | 素食平替：鸡胸肉→豆腐+毛豆+鸡蛋组合，保证总蛋白达标 |
| 过敏 | "我对牛奶过敏" | 过敏平替：牛奶→豆浆/杏仁奶 |

> 参照表内置10种蛋白质源、8种碳水源、5种脂肪源的营养密度数据。

### 🏋️ 训练动作平替

当用户受设备限制、场地限制或伤病需要换动作时，模型按**动作模式 + 难度 + 设备**三个维度自动匹配合适的动作：

1. **确定动作模式**：从六大模式（水平推/垂直推/水平拉/垂直拉/膝主导蹲/髋主导铰链）中找到目标模式
2. **确定难度**：初级（自重）→ 中级（负重）→ 高级（爆发力）
3. **确定设备**：杠铃哑铃 / 弹力带 / 纯自重

最终从 6动作模式 × 3难度 × 3设备 = **54个组合**中选出最合适的动作。

| 使用场景 | 示例指令 | 技能自动执行 |
|:---------|:---------|:-------------|
| 没器械 | "我在家，只有弹力带" | 查弹力带列，从对应模式中选弹力带动作为替换 |
| 新手 | "我是新手，动作简单点" | 选该设备下的「初级」难度动作 |
| 进阶 | "俯卧撑太简单了" | 升级到「中级」或「高级」同模式动作 |

## 🔧 自定义指南

1. **修改系数**：编辑 `SKILL.md` 中的蛋白质系数（如减脂期 2.0 → 2.2）
2. **替换训练模板**：编辑 `SKILL.md` 5.3 节中的周计划表格
3. **调整压力/睡眠阈值**：修改 `SKILL.md` 中压力水平 >7 或睡眠 <6h 的判断条件
4. **扩展营养参照表**：在 `SKILL.md` 第二步的「常用食物营养参照表」中添加更多食物数据
5. **扩展动作平替表**：在 `SKILL.md` 5.4 节中添加更多动作变体
6. **扩展微量营养素关联**：在 `SKILL.md` 第六步中增加更多自查项
7. **增加参数**：在 `SKILL.md` 第一步的参数表中添加新字段
